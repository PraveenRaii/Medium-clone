# Medium Clone

A pixel-inspired clone of the Medium homepage created using pure HTML and CSS.

## Features
- Responsive Header Navigation
- Search Bar
- Left Sidebar Menu
- Featured & For You Tabs
- Article Feed with Images and Stats
- Staff Picks Section
- Recommended Topics
- Sticky Header and Sidebars
- Clean and Modern UI

## Technologies Used
- HTML5
- CSS3
- js

## Purpose
This project was built to practice frontend development, CSS layouts, Flexbox, positioning, and recreating real-world website designs.
